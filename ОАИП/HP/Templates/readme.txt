====================
Files in this folder
====================

This folder contains templates for the various output formats of 
Help & Manual. The folder structure is organized as follows:

\Templates\HTML Skins

	*.hmSkin        HTML skins for export with skins

\Templates\Webhelp

	*.zlang         Localized message strings for full-text search
	                in browser-based help. 

\Templates\PDF

	*.mnl           Print manual templates for print and PDF output

\Templates\Word

	*.docx		Templates for Microsoft Word output with placeholder tags


\Templates\Translation

 	*.*		Configuration files for SDL Trados and Access
